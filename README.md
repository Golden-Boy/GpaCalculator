# Grade Point Average Calculator
GPA Calculator for Solano College TRIO/Math department. The following application was developed using PyQt4 GUI framework and PyQt Designer to seamlessly implement a functional user interface.

The various tabs of the application allows the students to do the following:

	Determine the needed grade to end course with intended grade
	Calculate grades by semester

# Requirements
python3

PyQt4

# How to Run
python3 main.py
